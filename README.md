## How To Start the Genetic Algorithm

### Requirements

- python        3.7.3
- pygame        1.9.6
- numpy         1.16.2
- matplotlib    3.0.3 

If you have all the requirements installed, you can run the program in the command prompt / terminal.

```bash
cd Space_Invaders
python ga_player.py
```
